
Contents of the package:

1. A test dataset containing 1000 samples (Test_1000)

    20 chunk_xx.pt files, each containing 50 matrix samples.
   One metadata.pt file containing the dataset format and metadata information.

2. Four trained models (model_x.pth) and their corresponding test results (test_results for model_x.pth.json)

   The test results are generated by running the testing code and include evaluation metrics such as Accuracy, Recall, Precision, F1-score, and other performance metrics.

3. Executable testing code (Model_test.py) and the network implementation (net/nn24.py)
